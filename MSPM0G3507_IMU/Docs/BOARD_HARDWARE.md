## 4. 建议的底层驱动分层

- `generated/syscfg/`：SysConfig 自动生成的时钟、GPIO 和外设初始化代码。
- `BSP/`：板级外设封装，如 UART、CAN、PWM、SPI/I2C 和 GPIO 的统一接口。
- `Drivers/`：陀螺仪和温度传感器等具体器件驱动。
- `User/`：底层驱动测试和应用逻辑。
- `Algorithm/`：姿态解算、温控算法等与硬件访问解耦的算法代码。

## 6. 调试与安全注意事项

- 加热电路首次测试前应确认 MOSFET 驱动极性；初始化阶段默认关闭 PWM 输出。
- 加热测试应设置占空比上限、目标温度上限和传感器异常关断逻辑。
- CAN 总线两端需要正确配置终端电阻，并确认收发器供电和逻辑电平。
- UART 连接前确认 TX/RX 交叉连接、共地和电平兼容性。
- IMU 和温度传感器上电后优先读取器件 ID 或状态寄存器，再开始连续采样。
