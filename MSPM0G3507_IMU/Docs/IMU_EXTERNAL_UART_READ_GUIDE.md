# IMU 对外 UART 读取说明

串口使用 `460800`、8N1。读取端按字节接收并持续解析，不能假设一次串口回调恰好得到一整帧。

## 读取自定义格式

1. 扫描帧头 `A5 5A`，读取 `LEN`。
2. 再收满 `LEN + 2` 个字节（正文和 CRC）。总长度为 `LEN + 5`。
3. 对 `LEN` 到 payload 末尾计算 CRC16-CCITT；不一致则丢弃当前帧并重新找 `A5 5A`。
4. 仅在 `TYPE=03、ID=01、LEN=16` 时按 20 字节数据 payload 解析。

所有数据为小端序。角速度除以 `1000` 得 `deg/s`；角度除以 `1000` 得 `deg`。先检查 `flags.bit2`：为 0 时当前角度尚未完成本次上电的校准，不能当作实际姿态使用。

接收命令响应时，检查 `TYPE=02`，其 `ID` 就是命令号，payload 第 0 字节是响应状态。`STATUS=00` 仅表示设备已接收请求，校准结果以后续数据帧的 `calResult` 为准。

## 读取 JY901 格式

1. 扫描 `0x55`，再收满固定 11 字节。
2. 对前 10 字节做 SUM8，并与第 11 字节比较；失败则重新扫描帧头。
3. `TYPE=0x52` 时读取第 6、7 字节（`int16` 小端）为 Z 轴角速度。
4. `TYPE=0x53` 时读取第 6、7 字节（`int16` 小端）为航向角。

换算：`角速度 = raw / 32768 * 2000 deg/s`，`角度 = raw / 32768 * 180 deg`。`0x52` 与 `0x53` 是两帧独立数据，接收端分别缓存后显示即可。JY901 格式没有校准有效标志；本固件会在本次上电校准成功后才开始发送它。

JY901 模式下，如需触发校准或归零，仍向同一串口发送自定义 `A5 5A` 命令帧，并按自定义格式读取响应帧。
